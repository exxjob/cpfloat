int free_optstruct(optstruct *fpopts)

Free the memory underlying an optstruct struct.

This function attempts to free all the memory used by fpopts.

Parameters:

fpopts – [in] Pointer to optstruct struct to be deallocated.

Returns:

The function returns 0 if the unless fpopts is set to NULL, in which case it return -1.
